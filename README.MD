## Загрузка и предобработка данных

   - `Color_ID` на `Int64.Type`

   - `Country_ID` на `Int64.Type`

### Dimensions.csv
1. **Замененное значение:** Заменяем точку на запятую в столбцах, где она используется как десятичный разделитель.
2. **Измененный тип:** Изменяем типы данных соответствующих столбцов на целочисленные и числовые типы:
   - `Clearance`, `Wheelbase`, `Height`, `Length`, `Width` на `Int64.Type`

## Создание мер

Добавлены следующие меры:

### Средняя стоимость авто
```DAX
Средняя стоимость авто = SUM(Car_General[Price]) / COUNT(Car_General[Car_ID])
```

### Средняя стоимость ЛС
```DAX
Средняя стоимость ЛС = SUM(Car_General[Price]) / SUM(Engine[Maximum_power])
```

### Среднее время жизни модели
```DAX
Среднее время жизни модели = YEAR(MAX(Car_General[End_Serial])) - YEAR(MAX(Car_General[Release_start_series]))
```
